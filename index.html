Welcome to MHGIS!

This is a tool to explore data from different spatial data sets about Malaysia.

Each color on this current map represents one data set which you can explore by clicking on the content tab with the same color.

Roster of Malaysian Chinese Associations

(马来西亚华团总名册)

This dataset is was adapted from a book named "马来西亚华团总名册" (Roster of Malaysian Chinese Associations), published by 马来西亚中华大会堂总会（华总）(The Federation Of Chinese Associations Malaysia) and 星洲日报 (Sin Chew Daily) in 2005. It documents most ethnic Chinese associations and groups them by state and by type of association.

Use control-f to search through points which are currently on screen. Click on the corresponding card below to highlight the desired point.
